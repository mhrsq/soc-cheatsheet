Install: Wazuh Agent

Hands-on 08

Apa yang dilakukan Wazuh Agent?

  • Monitor proses yang berjalan
  • Track koneksi jaringan
  • Deteksi perubahan file penting
  • Kumpulkan log OS & aplikasi
  • Kirim semua ke SIEM dashboard
Milestone: Nama laptop kalian muncul di Wazuh Dashboard dengan status ● Active

Steps

  1. Buka cheat sheet → Wazuh → Getting Started
  2. Klik section "Install Wazuh Agent"
  3. Pilih tab OS kalian (Linux / Windows / macOS)
  4. Ganti [MANAGER_IP] dengan IP yang gua kasih
  5. Jalankan semua command
  6. Cek di Dashboard → Agents

⏱ Estimasi: 15-20 menit termasuk troubleshoot

🦈

Install: Wireshark

Hands-on 09

Network Packet Analyzer

Capture semua packet di network interface. Lihat data mentah yang lewat — dari HTTP request sampai DNS query.

Quick test: Start capture → buka browser → stop → lihat packet list. Itu network traffic kalian.

Steps

  1. Buka cheat sheet → Wireshark → Install
  2. Pilih tab OS
  3. Install & restart jika diminta
  4. Open Wireshark → pilih network interface
  5. Start capture 30 detik → Stop
  6. Lihat hasilnya! 🎉

⏱ Estimasi: 10 menit

Live Demo: Trigger Alert

Hands-on 11

Sekarang kita buktikan bahwa semuanya connected. Gua akan trigger aktivitas mencurigakan, dan kalian lihat alert-nya muncul di Wazuh Dashboard.

🔴 Attack Simulation

# Port scan dari terminal
nmap -sV localhost

# Atau trigger brute force
ssh invalid@[VPS_IP]
ssh invalid@[VPS_IP]
ssh invalid@[VPS_IP]

🟢 What to See

  1. Buka Wazuh Dashboard
  2. Navigate ke Threat Intelligence → Alerts
  3. Filter: rule.level: >= 7
  4. Alert muncul dengan detail: source IP, timestamp, rule description
Ini yang terjadi setiap hari di SOC. Alert masuk, analyst investigate.
